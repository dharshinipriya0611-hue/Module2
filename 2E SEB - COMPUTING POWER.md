## SEB - Multiply the input

---

### AIM  
To write a Python program that accepts three integer inputs from the user, multiplies them, and displays the product.

---

### ALGORITHM
~~~

1.Start the program.
2.Read three integer values a, b, and c from the user.
3.Pass the values to the result(a, b, c) function.
4.Inside the function, calculate the product using mul = a * b * c.
5.Display the result (mul) and stop the program.
~~~

### PROGRAM

```
def result(a, b, c):
    mul =a*b*c
    print("Multiply is",mul)   
a=int(input())
b=int(input())
c=int(input())
```
